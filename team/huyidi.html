PROJECT: League Tracker

Overview

CS2113T Software Engineering module in Semester 2 AY 2018/2019 required us to do a team project of enhancing a basic command line interface application addressbook. My team and I decided to morph AdressBook Level 3 into a football league records management application called League Tracker. This enhanced application helps league managers to record relevant information of players, teams, matches and finances in the league. Moreover, it processes the data to provide league managers with better organised information. League Tracker is a CLI (Command Line Interface) application written in OOP fashion with a very basic GUI.

My role in this project was to design and write the codes for the finance feature. The following sections illustrate my contributions in more detail, as well as the relevant sections I have added to the user and developer guides in relation to the finance feature.

Summary of contributions

  • Major enhancement: added the finance feature

    • What it does: It allows league managers to keep track of the financial income of each team in the league and even visualise the trend of changes within certain time period.

    • Justification: This feature improves the product because the financial aspect of a league is important to league managers. The profitability of each team and the league indicates the survivability of them in the future.

    • Highlights: This enhancement processes existing data from other classes and requires data analysis. It implements visualisation of data trends as well.

Finance

The Finance Feature in the application helps you to manage and keep track of financial conditions in the league.

Get League Finance

You can use the Get League Finance command to view the total financial condition of the league.

Format : getlf

Example:
getlf

Figure 1. result after typing "getlf"

The histogram of financial incomes of four quarters helps you to better visualise the trend of changes of financial income.


Get Team Finance

You can use the Get Team Finance command to view the financial condition in USD of a selected Team in League Tracker.

Format : getfinance INDEX

Example:

  • Listteam

  • getfinance 4

Figure 2. the list of teams displayed after typing "listteam"
Figure 3. result of getting finance of the selected team

This Get Team Finance command enables you to view the financial condition of a team from the list of teams. This is applicable when you are viewing the team list and then you get curious about the financial condition of a certain team in the team list.


List Team Finance

You can use the List Team Finance command to view the list of financial condition in USD of all Teams in the league.

Format : listfinance

Example:
listfinance

Figure 4. result of listing finances

The order of the finance list corresponds to the order of the list of teams.


View Team Finance

You can use the View Team Finance command to view the financial condition in USD of a selected Team in League Tracker from the finance list.

Format : viewfinance INDEX

Example:

  • Listfinance

  • viewfinance 4

Figure 5. the list of finances displayed after typing "listfinance"
Figure 6. result of viewing finance of the selected team from the finance list

This View Team Finance command gives you a more detailed version of the financial condition compared to the result of Get Team Finance command. More information on sponsorship and ticket income is given.


Rank Team Finance

You can use the Rank Team Finance command to view a ranked list of financial incomes of all Teams from high to low.

Format : rankfinance

Example:
rankfinance

Figure 7. result of ranking finances
